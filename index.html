Halo! Ini halaman web sederhana.

Template ini cocok untuk portfolio, landing page kecil, atau latihan HTML/CSS. Mudah dikustomisasi.

Responsive

Tampil rapi di layar kecil dan besar tanpa framework.

Single file

HTML + CSS + sedikit JS — cukup satu file untuk mulai.

Mudah edit

Ganti teks, warna, atau gambar sekejap untuk disesuaikan.

Cara pakai singkat

  1. Buka file ini di editor (VS Code, Notepad++).
  2. Edit teks di <h1> dan paragraf.
  3. Ganti gambar dengan URL atau file lokal.

Kontak

Mau dimodif jadi portfolio atau ditambahkan form? Kasi tau aku.

Dibuat dengan ❤️ — buka file "Halaman Simple - index.html" untuk kodenya.
